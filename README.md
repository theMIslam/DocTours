# Doctour

```
cd existing_repo
git remote add origin https://gitlab.com/theMIslam/doctour.git - привязывает репозитории по той ссылке которая указана в GIT
git branch -M main - создаёт ветку по названию
git push -uf origin main - пушит в систему контроля версии GIT
```

```
git pull origin dev - стягивает то что есть на ветке dev
git push origin dev --force - пушит поверх то что есть в ветке или же в репозитории
git remote remove origin -- отвязывает от того репозитория к которому вы привязаны
git remote -v - можно посмотреть к какому репозиторию проект ваш привязан
git log - посмотреть все commit (история)
git push --help - помогает для того чтобы узнать какие есть команды и за что отвечает каждая команда
git pull - стянуть со всех веток которые есть в вашем репозитории
git fetch origin - обновление данных в GIT
git stash - локальная база данных
git status - просмотр статуса проекта в GIT
```

```
git init - инициализация к удалённому репозиторию которая ещё не привязана к GIT
git add . - выделяет все что вы сделали внутри проекта
git commit -m "comment" - поставить комментарии что вы сделали
git remote add origin https://gitlab.com/theMIslam/doctour.git - привязывает репозитории по той ссылке которая указана в GIT
git push origin main - пушит все изменения которые вы сделали
```
